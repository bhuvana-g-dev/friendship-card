# Special Card Frontend 🎁

A responsive  web application for creating personalized shareable greeting cards.

Users can add names, write messages, select avatars, generate a unique card link, copy it, and share instantly through WhatsApp.

---

## 🌐 Live Demo

https://friendship-card-ry3y.vercel.app

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Vercel

---

## ✨ Features

* Personalized card creation
* Avatar selection with animations
* Mobile responsive interface
* Unique shareable link generation
* Copy link functionality
* WhatsApp sharing
* Backend API integration
* Clean and simple UI
